Exemplo de Seletores

Seletor de TAG

Quando damos características a TAGs (como h1, h2), todos os elementos desse tipo que aparecer na estrutura HTML, vão herdar essas características.

Seletor de ID

O ID só pode ser usado uma vez em toda a estrutura, ele funciona como um CPF. É individual para cada elemento, e geralmente usado em JavaScript

Seletor de CLASSE

As classes, diferente do id, ela pode ser reutilizada em diversos elementos do site.
